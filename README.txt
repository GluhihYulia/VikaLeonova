В файле Fibonachi.py содержится код, который выводит все элементы последовательности Фибоначчи, расположенные на отрезке [a, b]. 
На вход программе подаются два целых числа: a и b. 
Программа выводит члены последовательности Фибоначчи, которые попадают в отрезок [a, b], включая концы отрезка. Например, если a = -3, b = 14, программа выведет 0 1 1 2 3 5 8 13.
Программа выводит сообщение “Ошибка”, если если a >= b. Например,  a = 7, b = 5.
Программа выводит сообщение "В заданном диапазоне нет чисел Фибоначчи", если если в отрезке не содержатся числа Фибоначчи. Например, a = 10, b = 11. Или a = - 7, b = - 5.


В файле bibliotheque.py содержится код, который позволяет работать с электронной библиотекой. При запуске программы появляется меню: 
1 Добавить книгу
2 Посмотреть список книг
3 Отсортировать книги
4 Удалить книгу
5 Добавить книгу в избранное
6 Удалить книгу из избранного
7 Выйти из библиотеки


Если пользователь вводит “7”, то происходит выход из программы. 


Если пользователь вводит ”1”, то программа предлагает ввести автора, название и жанр добавляемой книги. При этом эта информация добавляется в файл bibliotheque.txt


Если пользователь вводит “2”, то программа выводит список добавленных книг.
Если файла не существует, то программа выводит сообщение “У вас нет добавленных книг”


Пункты 3, 4, 5, 6 пока не реализованы.
